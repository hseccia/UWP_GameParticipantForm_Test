A side test-project to further experience .NET and Universal Windows Platform.

Author: Hannah Seccia
Date of Creation: December 2019

Description: 

A program that simulates a registration page for a video game tournament.
The user fills out a form that asks their username, last name and age.
The user then picks their main game they play in a combo box.
Below the form, a button is clicked to save the profile, confirming this action with speech.

A List View is shown below the button that displays every user entered and their info.


Bug List:
-If user tries to enter a letter/character in the age text control, the program crashes.
